About

Hi, this is Cong, thank you for visiting my website.

I'm a big fan of front-end technologies. building fantastic and visualized web applications is always my love and passion.

While watching "Iron Man", one idea came into my mind:

The flat UI design for those HUD in sci-fi movies / games is just amazing, but why I never saw one website has been built this way?

Because it's impossible? Limited by browser rendering engine's functionality? Or just no one wanna try it out?

Anyway, after some research online, I found it's doable and decided to use my free time to design and build a new website this way.

The task was not easy for me, many problems poped up, svg/canvas, css/javascipt animation control, 3D matrix calculation, etc. I managed to solve them one by one and finally got this prototype.

It's a wonderful experience cause I learnt and built something I really enjoy.

But is it finished? No, not even close.

There are many features I'm planning to add:

  • implementation of audio visualization
  • 3D space game using WebGL (ThreeJS maybe?) as background
  • mutiple 2D interactive animations using canvas/svg
  • fully responsive on mobile
  • a open source project of sci-fi ui

So, send me a message or email if you got any questions or ideas
